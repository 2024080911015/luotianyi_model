# 洛天依桌面助手 - 图片版

使用外部图片文件的洛天依桌面宠物应用。

## 快速开始

1. **安装依赖**:
```bash
pip install -r requirements.txt
```

2. **准备图片**:
   - 创建 `images/` 目录
   - 放入洛天依图片（见下文说明）

3. **运行程序**:
```bash
python main_image.py
```

## 图片文件要求

### 方案一：单张图片模式（推荐）
在 `images/` 目录下放入以下6张图片：
- `idle.png`     - 站立状态
- `walk.png`     - 行走状态  
- `sit.png`      - 坐下状态
- `sleep.png`    - 睡觉状态
- `happy.png`    - 开心状态
- `surprise.png` - 惊讶状态

**图片要求**:
- 格式：PNG（支持透明背景）
- 大小：建议 200x200 像素
- 背景：透明为佳

### 方案二：动画模式
创建子目录并放入多张图片形成动画：
```
images/
├── idle/      # 站立动画
│   ├── 01.png
│   ├── 02.png
│   └── ...
├── walk/      # 行走动画
├── sit/       # 坐下动画
├── sleep/     # 睡觉动画
├── happy/     # 开心动画
└── surprise/  # 惊讶动画
```

**动画要求**:
- 图片命名：按顺序，如 `01.png`, `02.png`, `03.png`
- 帧率：100ms/帧
- 格式：PNG（支持透明背景）

## 图片获取建议

1. **官方素材**:
   - VOCALOID 官方素材网站
   - 洛天依官方设定图

2. **AI生成**:
   - 使用 Stable Diffusion 生成Q版洛天依
   - 提示词示例：`chibi luotianyi, vocaloid, cute anime girl, blue hair, twin tails`

3. **同人作品**:
   - Pixiv、Twitter 等平台的同人图
   - 注意版权，仅限个人使用

4. **自己绘制**:
   - 使用绘图软件（SAI、Photoshop、Krita等）
   - 导出为PNG格式

## 如果没有图片

如果 `images/` 目录为空或图片缺失，程序会显示一个简单的圆形默认角色。

## 功能特点

- 🖼️ **支持外部图片**: 使用真实的洛天依形象
- 🎬 **可选动画**: 支持单张图片或动画序列
- 🖱️ **完全交互**: 拖拽、点击、右键菜单
- 💬 **智能对话**: 随机说话、时间问候
- 🎨 **透明窗口**: 始终置顶，不影响工作
- ⚙️ **可配置**: 调整大小、切换动画模式

## 使用方法

### 基本操作
- **左键拖拽**: 移动洛天依
- **右键点击**: 打开设置菜单
- **鼠标悬停**: 触发互动反应
- **ESC键**: 退出程序

### 右键菜单功能
- **说话**: 让洛天依随机说话
- **调整大小**: 小(150px)、中(200px)、大(250px)
- **切换动画**: 启用/禁用动画模式
- **关于**: 显示程序信息
- **退出**: 关闭程序

## 常见问题

### Q: 图片不显示或显示错误
**A**: 检查：
1. 图片格式是否为PNG
2. 图片文件名是否正确
3. 图片是否在正确的目录

### Q: 动画不流畅
**A**: 调整 `frame_delay` 参数（单位：毫秒）

### Q: 图片有白边
**A**: 使用图片编辑软件将背景设置为透明

### Q: 想使用不同风格的图片
**A**: 直接替换 `images/` 目录下的图片即可

## 从代码生成版迁移

如果你之前使用代码生成版，现在想用图片版：

1. 保留原有的 `main.py`（代码生成版）
2. 新增 `main_image.py`（图片版）
3. 两者可以共存，互不影响
